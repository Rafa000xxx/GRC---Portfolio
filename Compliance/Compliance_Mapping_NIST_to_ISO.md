### Regulatory Crosswalk: Identity & Access Management
This document maps NIST CSF 2.0 sub-categories to ISO 27001:2022 Annex A controls to ensure global compliance.

| NIST CSF 2.0 ID | Description | ISO 27001:2022 Control |
| :--- | :--- | :--- |
| **PR.AC-1** | Identities are managed and authenticated. | **A.5.16** (Identity Management) |
| **PR.AC-3** | Access is granted based on Least Privilege. | **A.5.18** (Access Rights) |
| **PR.AC-4** | Network access is managed (Remote Access). | **A.8.22** (Network Security) |

**Summary:** By enforcing MFA on our VPN (PR.AC-4), we satisfy both NIST and ISO requirements for secure remote network access.
